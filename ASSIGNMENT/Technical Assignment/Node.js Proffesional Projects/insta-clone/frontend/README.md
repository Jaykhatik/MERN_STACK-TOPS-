# Instagram Clone - Frontend

This is the frontend application for the Instagram Clone project. It is a single-page application built with React, TypeScript, and Vite, featuring a modern and responsive user interface.

## 🚀 Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: Adds static typing to JavaScript for better developer experience and code quality.
- **Vite**: Next-generation frontend tooling for fast development and build times.
- **Vanilla CSS**: Used for styling the application, ensuring maximum flexibility and a custom, premium design.

## 🎨 Features

- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices.
- **Modern UI**: Implements a sleek, user-friendly interface inspired by Instagram, featuring dynamic interactions and smooth animations.
- **Post Feed**: View a continuous feed of posts from other users.
- **Create Post**: Modal-based post creation.
- **Interactions**: Like and comment on posts.
- **User Profiles**: View user profiles and their posts.

## 📁 Project Structure

```text
src/
├── assets/         # Static assets like images
├── components/     # Reusable React components (e.g., PostCard, Stories, Navbar)
├── pages/          # Top-level page components
├── App.tsx         # Main application component
├── index.css       # Global styles
└── main.tsx        # Entry point
```

## 🛠️ Getting Started

### Prerequisites

Ensure you have Node.js installed.

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Build

To create a production-ready build:

```bash
npm run build
```

The built files will be output to the `dist` directory.
